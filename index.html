About Me

February 2016 I took my first glimpse behind the tech world curtain. I started taking an online Ruby course. 14 hours and one forgotten meal later, I was fascinated. Overnight I became a diver with a serious coding problem. My newfound obsession took over and I dove (pun intended) in deep. Now I'm here, with a handle on Rails, JavaScript, React, Redux, SQL, Vue, and a knack for styling. I love the challenge of learning new languages and frameworks and then applying them to build and design something real. I am currently looking for a frontend role in San Francisco, and am so excited to contribute to a company that is doing some good in this world.

Projects

AirGoT is an Airbnb inspired single page app with a Game of Thrones theme. It has Rails backend, a PostgreSQL database, and a JavaScript/React/Redux frontend. Users can search all of Westeros and Essos using a Google Maps API with a custom map overlay, as well as make a booking, write reviews, and check out profiles of their favorite characters.

Hoo is a single page app designed as a staffing solution for short-term work in the service industry. It has a Rails backend with a Vue front-end using VueX and Vue Router, and is hosted on AWS with an EC2 instance using Ubuntu and Apache.

Hungry Sharks is a memory game designed to increase the user's ability to multi-task. Built only using Vanilla JavaScript and HTML5 Canvas, it tracks the circling sharks, the number of fish left to feed them, and game state. It also provides the user with instructions, a timer, and controls to pause and mute.
